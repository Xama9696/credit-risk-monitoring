# Credit Risk Model Monitoring System

This project simulates an end-to-end model monitoring pipeline used in financial institutions to track the performance and reliability of credit risk models.

A dataset of loan predictions is processed using Python to generate daily monitoring metrics such as model accuracy, approval rate, average income, and loan amount. The pipeline automates data cleaning, feature engineering, and aggregation to create a structured monitoring dataset.

The processed data is then visualized using Tableau in an interactive dashboard. The dashboard highlights trends in model accuracy over time, approval rate fluctuations, and key performance indicators, enabling better monitoring and decision-making.

This project demonstrates how machine learning systems are not only built but also continuously monitored in production environments.

## Tech Stack

* Python (Pandas, NumPy)
* Jupyter Notebook
* Tableau

## Key Features

* Automated model monitoring pipeline
* Daily performance tracking (accuracy, approval rate)
* Data pipeline → dashboard workflow
* Business-focused analytics visualization

## Output

* Monitoring dataset (`monitoring_dashboard.csv`)
* Tableau dashboard with trends and KPIs
